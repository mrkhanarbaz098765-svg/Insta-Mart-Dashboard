# 📊 Swiggy Instamart Sales Dashboard | Power BI

## 📌 Overview
An interactive Power BI dashboard created to analyze Swiggy Instamart sales performance, customer trends, shop types, and category-wise insights.

---

## 🚀 Features
- Total Sales & Orders Analysis
- Avg. Sales & Customer Rating
- City Tier Performance
- Category-wise Sales Breakdown
- Shop Type & Size Insights
- Interactive Filters & KPIs

---

## 🛠️ Tools Used
- Power BI
- DAX
- Power Query
- Data Visualization

---

## 📈 Key Insights
- 💰 Total Sales: **1.20M+**
- 📦 Total Orders: **9K**
- ⭐ Avg. Rating: **3.92**
- 🏙️ Tier 3 cities generated highest sales
