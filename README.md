📌 Project Overview

This Power BI Dashboard project analyzes Swiggy Instamart sales performance using interactive visualizations and KPI tracking. The dashboard helps in understanding sales trends, customer behavior, city-tier performance, and category-wise analysis for better business decision-making.

🚀 Dashboard Features
📈 Total Sales Analysis
🛒 Total Orders Tracking
⭐ Average Customer Rating
🏙️ City Tier Performance Analysis
🏪 Shop Type & Shop Size Insights
📅 Year-wise Shop Opening Trend
🥗 Category-wise Sales Breakdown
🎛️ Interactive Filters & KPI Cards
🛠️ Tools & Technologies Used
Microsoft Power BI
DAX (Data Analysis Expressions)
Power Query
Data Cleaning & Transformation
Business Intelligence Reporting
Data Visualization
📊 Key Insights
💰 1.20M+ Total Sales analyzed
📦 9K Total Orders tracked
⭐ 3.92 Average Customer Rating
🏙️ Tier 3 cities generated the highest sales
🍎 Fruits & Vegetables category contributed the most sales
